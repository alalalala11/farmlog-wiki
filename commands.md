# ⌨️ 명령어 안내

자주 사용하는 명령어 목록입니다.

---

## ✈️ TPA (텔레포트 요청)

| 명령어 | 설명 |
|---|---|
| `/tpa <닉네임>` | 해당 플레이어에게 텔레포트 요청 (30초 유효) |
| `/tpaccept` | 텔레포트 요청 수락 |
| `/tpdeny` | 텔레포트 요청 거절 |

---

## ✉️ 귓속말

| 명령어 | 설명 |
|---|---|
| `/귓말 <닉네임> <메시지>` | 귓속말 보내기 |
| `/r <메시지>` | 마지막 귓속말 상대에게 답장 |

---

## 💰 돈 랭킹

| 명령어 | 설명 |
|---|---|
| `/돈랭킹` | 서버 잔액 TOP 10 조회 |

---

## 🐄 동물 관련

| 명령어 | 설명 |
|---|---|
| `/동물목록` | 내 목장 보유 동물 현황 |
| `/동물부르기 <종류>` | 내 동물을 근처로 소환 (소/닭/양/돼지/전체) |
| `/동물양도 <닉네임>` | 바라보는 동물을 해당 플레이어에게 양도 요청 |
| `/동물수락` | 동물 양도 요청 수락 |
| `/동물거절` | 동물 양도 요청 거절 |

## 서버 소개

목장일기는 **농사, 목축, 채광**을 중심으로 한 소규모 경제 RP 서버입니다.

처음 접속하면 **길라잡이**가 시작되어 서버 생활의 기초를 안내해드립니다.  
마을에서 NPC들과 상호작용하고, 목장을 가꾸고, 광산에서 채굴하며 경제 활동을 즐겨보세요.

---

## 주요 콘텐츠

- 🏡 **목장 시스템** — 닭·소·양·돼지 사육, 감별기로 등급 부여, 합성
- ⛏️ **광산** — 광산 1(석탄·구리·철), 광산 2(금·레드스톤)
- 🌾 **농사** — 밀·감자·당근·호박·수박 등 작물 재배 및 가공
- 📚 **특성책** — 생활·채광·목축·농사 특성을 구매해 활동 범위 확장
- 🏪 **상점** — NPC 상점에서 아이템 구매·판매
- 🗓️ **보부상** — 요일별 스페셜 아이템 판매
